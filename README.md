### Clone repository

```bash
$ git clone https://github.com/souzaluuk/db-corona/
$ cd db-corona
```

### Install virtual environment

```bash
$ python3 -m venv env
$ source env/bin/activate
(env) $ pip install -r requeriments.txt
```

### Run tests

```bash
(env)$ cd db
(env)$ python -m unittest -v tests
```
